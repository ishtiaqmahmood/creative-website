# Creative Website - Modern Landing Page

A professional, responsive, and interactive landing page designed for creative agencies and businesses. Built with HTML5, CSS3, Bootstrap 4, and jQuery.

## Features

- **Modern UI/UX Design:** Clean and professional look with a vertical heading style.
- **Fully Responsive:** Optimized for all screen sizes, from mobile to desktop.
- **Interactive Elements:**
  - Smooth scrolling navigation.
  - Dynamic navbar background transition on scroll.
  - Custom preloader for a smooth user experience.
- **Key Sections:**
  - **Hero Section:** Engaging introduction with a call-to-action.
  - **About Section:** A clean layout showcasing the story behind the brand.
  - **Services Section:** Highlighting core offerings with interactive cards.
  - **Stats Section:** Visual representation of key metrics.
  - **Portfolio Section:** A grid layout to display past work.
  - **Team Section:** Meet the professionals behind the work.
  - **Testimonials:** Feedback from happy clients.
  - **Contact Section:** Integrated contact form for easy communication.

## Technologies Used

- **HTML5:** For structured content.
- **CSS3:** Custom styling and parallax effects.
- **Bootstrap 4:** For the responsive grid and components.
- **jQuery:** For smooth animations and interactive features.
- **FontAwesome:** For intuitive iconography.
- **Google Fonts:** Raleway and Open Sans for professional typography.

## Setup and Usage

To run this project locally:

1. Clone the repository to your local machine.
2. Open `index.html` in any modern web browser.

## Project Structure

- `index.html`: The main landing page structure.
- `css/`:
  - `style.css`: Custom project styles.
  - `bootstrap/`: Bootstrap framework files.
- `js/`:
  - `script.js`: Interactive project logic.
  - `jquery.js`: jQuery library.
- `img/`: Organized assets for various sections of the page.
